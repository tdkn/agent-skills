# Agent Skills

[English](README.md) | **日本語**

AI コーディングエージェントの機能を専門知識とワークフローで拡張するためのスキル定義集です。

## 概要

このリポジトリには、AI エージェントが特定のツール、API、ワークフローをいつ、どのように使用するかを理解するのに役立つスキル定義が含まれています。各スキルは以下を提供します：

- 明確なユースケースとトリガー
- ステップバイステップのワークフロー
- API パターンと例
- ベストプラクティスとトラブルシューティングのヒント

## 構造

```
agent-skills/
└── skills/
    └── sosume/
        └── SKILL.md    # Apple ドキュメントを取得するための Sosumi スキル
```

## スキル

### [Sosumi](./skills/sosume/SKILL.md)

Sosumi 経由で Apple ドキュメントを Markdown として取得します。以下の用途に必須：

- Apple プラットフォーム API（Swift、SwiftUI、UIKit、AppKit、Foundation など）
- ヒューマンインターフェイスガイドライン
- WWDC セッショントランスクリプト
- 外部 Swift-DocC ドキュメント

**クイックスタート**: `developer.apple.com` の URL を `sosumi.ai` に置き換えることで、Apple ドキュメントの AI 可読な Markdown バージョンを取得できます。

## 使い方

各スキルは、独自のディレクトリ内の `SKILL.md` ファイルで定義されています。スキルは標準的な形式に従います：

- **フロントマター**: 名前と説明
- **使用すべき場合**: スキルが適用される明確なトリガー
- **コアワークフロー**: ステップバイステップのプロセス
- **例**: 具体的な使用パターン
- **ベストプラクティス**: 最適な結果を得るためのガイドライン
- **トラブルシューティング**: よくある問題と解決策

## コントリビューション

新しいスキルを追加するには：

1. `skills/` 配下に新しいディレクトリを作成
2. 確立された形式に従って `SKILL.md` ファイルを追加
3. この README のスキルセクションに簡単な説明を追加して更新
